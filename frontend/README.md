# Folder Structure Overview

```bash
/src
├── /assets           # static assets like images, icons, logos
├── /contexts         # context providers
├── /components       # reusable UI components (buttons, layout blocks, etc.)
├── /features         # domain-based folders (Books, Tags, Users, etc.)
│   ├── /books
│   └── ...
├── /hooks            # custom React hooks
├── /lib              # utils, API clients, schemas
├── /mobile           # mobile specific platform code (for Android/IOS)
├── /pages            # route-level pages (BooksPage, HomePage, etc.)
├── /routes           # React Router config
├── /styles           # custom MUI theme, palette
└── /utils            # utility functions and constants
```
